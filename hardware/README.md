# Hardware

Files for the reference build — STLs, wiring diagrams, and a bill of materials — will
land here. They aren't published yet; the software runs on any host that can reach a
Music Assistant server, and drives a touchscreen in kiosk mode on a Raspberry Pi.

See [`../PROJECT.md`](../PROJECT.md) for the project overview and [`../INSTALL.md`](../INSTALL.md)
for the appliance install.
